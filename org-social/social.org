Haven't really used it so can' say what's wrong with it.
I can say however what is great about Magit.

While Magit makes complex things with Git quite accessible, it doesn't really dumb them down as all of the other programs do.
Sometimes with GUIs you have to go back to the CLI to do things, with Magit this hasn't been the case yet for me.
It shows you the CLI options you can apply 1:1 so it actually teaches you stuff while your using it.
In addition the key chords are amazing and it sometimes feels like an extension of your brain because you can do things so fast.

Sure if there are other programms that can do the same things as Git can do but in a better way that is a good thing to have and I'm curios how for example Jujutsu evolves.
However they will have to match the level of productivity I get with Magit before I consider switching.
